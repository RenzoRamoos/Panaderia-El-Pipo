Preguntas frecuentes

¿De que se trata el Proyecto?

Este proyecto comenzado a mediados de Septiembre, se nos ocurrió de hacer una panaderia, vender cada producto y tratar de que sea lo mejor posible para la vista de las personas.

¿Como se nos ocurrió?.

Este proyecto se nos ocurrió ya que estuvimos varias semanas hablando y no llegabamos a un acuerdo, un dia pensamos miles de opciones y llegamos a la panaderia y fue buena opción para todos.

¿Cual es nuestro objetivo de la panaderia?

El objetivo es Satisfacer a todos los consumidores generando variedades de productos y vender nuestro producto a un precio accesible para la comunidad.
